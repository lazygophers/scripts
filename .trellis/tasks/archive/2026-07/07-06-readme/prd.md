# PRD：精简 README 为纯使用手册

## 背景

现 README 混合使用手册 + 开发指南（目录结构/加新脚本/测试/贡献）。用户要纯使用手册。

## 目标

README 仅保留面向使用者的内容，删除开发向章节。

## 保留章节

1. **标题 + 简介**（首段，1-2 句）
2. **安装**（inject 注入 PATH + --show/--uninstall）
3. **功能表**（脚本功能表 + 示例 + 迁移说明）

## 删除章节

- 目录结构（调用链说明）
- 环境依赖
- 加新脚本（含别名脚本）
- 测试
- 参与贡献

## 验收标准

- README 仅含：简介 + 安装 + 功能表。
- 功能表内容完整不丢（含 push_* 批量说明 + 迁移说明）。
- 无失效链接/引用。
- 无开发向内容残留（lib/commands、unittest、PR 等）。
