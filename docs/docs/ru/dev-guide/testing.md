# Тестирование

```bash
python3 -m unittest discover -s tests -q
```

Набор тестов находится в `tests/` в корне репозитория. Для новых команд рекомендуется добавлять соответствующие модульные тесты, поддерживая раздельную гранулярность тестирования лёгкого входа и бизнес-логики (бизнес-логика тестируется под `lib/commands/`, лёгкие входы только передают данные и не требуют отдельных тестов).
