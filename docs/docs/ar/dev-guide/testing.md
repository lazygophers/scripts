# الاختبار

```bash
python3 -m unittest discover -s tests -q
```
مجموعة الاختبارات موجودة في `tests/` في جذر المستودع. يُنصح بإضافة اختبارات وحدة مقابلة للأوامر الجديدة، مع الحفاظ على تدرج الاختبار المنفصل بين المدخلات الخفيفة ومنطق الأعمال (منطق الأعمال يُختبر تحت `lib/commands/`، المدخلات الخفيفة تنقل البيانات فقط ولا تتطلب اختبارات منفصلة).
