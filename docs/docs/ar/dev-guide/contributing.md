# المساهمة

- يجب أن تتضمن طلبات السحب الاختبارات الوحدية اللازمة.
- النصوص البرمجية الجديدة تتبع خطوتين [« إضافة نص برمجي »](./add-script.md)، مع الحفاظ على الفصل بين المدخلات الخفيفة ومنطق الأعمال.
- القدرات المشتركة توضع في `lib/{مجال}.py`، قابلة لإعادة الاستخدام عبر الأوامر، وتجنب تكرار الكود في المدخلات الخفيفة أو الأوامر الفردية.
- عمليات Git تستخدم `git` القياسي، وتتجنب الأغلفة التفاعلية؛ قبل الأتمتة، إلزامية فحوصات نظافة شجرة العمل والتراجع.
